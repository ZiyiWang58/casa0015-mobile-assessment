Overview

PawTrack is a mobile app designed for dog owners who may forget daily walks or want a clearer view of their pet’s routine. The app combines dog management, walking goals, weather support, route tracking, summaries, history, reminders, and account-based cloud storage in one simple flow.

Core Features

Dog Profiles

Create multiple dog cards, save key details, and set a walking goal for each dog.

Live Walk Tracking

Track a walk in real time with GPS, route display, distance, duration, and calorie estimates.

Walk Summary

Finish a walk and instantly review results, including progress toward the goal.

History & 7-Day Stats

Review previous walks, compare recent activity, and understand routine over time.

Calendar & Reminders

Check whether a dog was walked on a given day and use reminders to stay consistent.

Per-User Cloud Data

Each account keeps its own dogs and walk records separated through Firebase login and Firestore.

API & Service Integration

Open-Meteo API

Used to fetch live weather data and generate walking advice based on current conditions.

Firebase Authentication

Used for user registration, login, logout, and account-based access.

Cloud Firestore

Used to store dog profiles and walk records in the cloud with per-user data separation.

Get Started

Release Build

  1. Open the GitHub release page.
  2. Download the latest app-release.apk.
  3. Install it on an Android device.
  4. Create an account or log in to start using PawTrack.

Run From Source

  1. Clone the repository.
  2. Run flutter pub get.
  3. Configure your own Firebase project.
  4. Enable Email/Password Authentication and Cloud Firestore.
  5. Run the app with flutter run.

Try PawTrack

Download the release build for quick testing, or open the GitHub repository to explore the project source, screenshots, and documentation.
